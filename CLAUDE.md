# Project Context & Rules
- **Framework:** LÖVE (Love2D) 
- **Language:** Lua
- **Graphics:** Use the asset `@your_asset.png`.
- **Hardware:** RTX 3090 Ti (Optimize for high FPS, use `dt` for all movement).
- **Environment:** Ubuntu Linux.

# Automation Rules
- Always run `love .` to test changes after editing code.
- Do not ask for permission to create folders or install basic Lua dependencies.
- If a bug is found during execution, fix it immediately without asking.
- Use modular file structures (e.g., `player.lua`, `map.lua`).
